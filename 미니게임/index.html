React Native 학습 결과물: Up & Down 미니게임

React Native 학습을 시작한 지 얼마 되지 않았지만, 이번에 간단한 Up & Down 미니게임을 완성했습니다. 이 앱은 사용자가 숫자를 입력하고, 기기가 그 숫자를 맞출 때까지 추측하는 게임입니다. 이번 포스팅에서는 프로젝트를 진행하면서 겪은 과정과 배운 내용을 공유하려고 합니다.

프로젝트 개요

이번 프로젝트는 숫자 맞추기 게임으로, 사용자가 입력한 숫자를 기준으로 기기가 숫자를 높이거나 낮추며 정답을 찾을 때까지 계속 진행하는 방식입니다. 학습을 위해 구현한 미니게임이지만, 기본적인 상태 관리, 화면 구성, 그리고 UI 로직 구현을 경험할 수 있는 좋은 프로젝트였습니다.

사용된 기술 스택

  • React Native

    : 모바일 애플리케이션을 개발하기 위한 프레임워크로, 자바스크립트와 React로 작성할 수 있어 학습하기 좋습니다.

  • Expo

    : 빠른 개발 환경 구축을 위해 Expo를 사용하였습니다. Expo는 번들링, 빌드 및 테스트 등을 빠르게 설정할 수 있어 프로젝트 초기에 유용합니다.

  • expo-font

    : 커스텀 폰트를 로딩하여 앱의 UI를 꾸미는 데 사용했습니다.

  • expo-splash-screen

    : 앱이 로딩될 때 Splash Screen을 표시하여 폰트가 준비될 때까지 유저 경험을 개선하였습니다.

주요 기능

  1. 숫자 입력 및 게임 시작

    : 사용자가 임의의 숫자를 입력하면, 그 숫자가 상태로 저장되고, 기기가 숫자를 맞추기 위한 추측을 시작합니다.

  2. 업앤다운 로직

    : 기기는 사용자의 숫자를 기준으로 더 높거나 낮은 숫자를 추측하며, 사용자가 입력한 값과 맞을 때까지 추측을 계속합니다.

  3. 게임 종료

    : 기기가 숫자를 맞추면 게임이 종료되고, 결과 화면이 표시됩니다.

배운 점

이번 프로젝트를 통해 React Native와 Expo 환경에서의 개발 과정을 이해할 수 있었습니다. 상태 관리, 이벤트 처리, 화면 전환 등의 핵심 요소를 다루면서 모바일 앱 개발의 기초를 쌓을 수 있었고, 특히 Expo를 사용한 폰트 로딩과 Splash Screen 구현 과정에서 앱의 퍼포먼스를 고려한 개발 방법을 학습했습니다.또한, React와 매우 유사한 구조 덕분에 학습에 큰 어려움이 없었고, 자연스럽게 앱 개발 환경에 대한 즐거움도 느낄 수 있었습니다. React와 공유하는 컴포넌트 기반 아키텍처 덕분에 익숙한 방식으로 작업할 수 있었고, 빠르게 결과물을 확인할 수 있다는 점이 특히 좋았습니다.React Native에서는 플랫폼별로 .jsx 파일을 만들어서 작업할 수 있고, Platform API와 같은 훅을 사용하여 간단하게 플랫폼 구분 로직을 처리할 수 있었습니다. 또한, KeyboardAvoidingView와 useWindowDimensions를 활용해 화면 크기에 따른 반응형 레이아웃을 쉽게 구현할 수 있어 개발 과정에서 많은 편리함을 느꼈습니다.앱 개발을 하다 보니 핸드폰에서 많이 사용하는 여러 앱들의 가로모드, 세로모드, 그리고 페이지 전환과 같은 스타일 차이가 눈에 더 잘 보였고, 각 플랫폼별 스타일 문제를 어떻게 해결했는지 궁금증이 생겼습니다. 이러한 학습 경험은 단순한 기능 구현뿐만 아니라 다양한 기기에서 사용자 경험을 향상시키기 위한 고민의 시작점이 되었습니다.

다음 계획

앞으로는 더 복잡한 상태 관리와 API 통신 기능을 추가하여 사용자와의 상호작용이 많은 앱을 개발할 계획입니다. 특히, 아직 useNavigation 훅을 공부하지 않았기 때문에, 두 화면 사이의 네비게이션 구현과 하단 탭 기능, 그리고 드로어 내비게이션 등의 UI 요소를 학습하고 적용할 예정입니다. 이러한 내비게이션 기능은 앱 사용성을 높이는 데 매우 중요한 부분이라, 이를 이해하고 실습하는 것이 중요하다고 느꼈습니다.

또한, 카메라, 위치, 사진 권한, 지도 등과 같은 네이티브 기능도 학습하여, 실제로 앱에서 이런 기능을 사용하는 방법을 익힐 것입니다. 궁극적으로는 이러한 학습을 바탕으로 실제 앱을 배포하는 것이 목표입니다. 이를 통해 React Native의 강력한 기능을 모두 활용해보고, 사용자가 실제로 사용할 수 있는 완성도 높은 앱을 만들고 싶습니다.

이번 학습을 통해 개발 프로세스 전반에 걸쳐 성장하는 경험을 쌓고, 실제 서비스에 가까운 프로젝트를 목표로 나아갈 것입니다.
